AI-assisted neuro-imaging review · concept build

A first look at what MRI screening review could feel like.

NeuroScan is a front-end concept for a five-step scan review workflow — patient intake, upload, simulated analysis, results and a printable report. Every output on this site is generated in your browser for demonstration purposes.

The workflow

Five steps, start to report.

Each step hands data to the next — patient details travel with the scan, the scan travels into the analysis stage, and the analysis output becomes both the result view and the printable report.

01 — INTAKE

Patient information

Capture the demo patient's identifiers and scan context before a file is attached, the way an intake form would.

02 — UPLOAD

Attach the MRI

Drop in an image exported from a scan series. The interface reads its metadata and shows a preview before continuing.

03 — ANALYSIS

Simulated processing

A staged progress sequence stands in for segmentation and feature-scoring steps a real pipeline might run.

04 — RESULT

Pattern summary

A mock scorecard renders region, pattern score and confidence — clearly labelled as illustrative, not diagnostic.

05 — REPORT

Printable report

The same figures are laid out as a clinical-style report sheet, ready to print or save as a reference document.

ALWAYS ON

History log

Every completed pass is logged locally so the workflow can be revisited without losing earlier runs.

Why this exists

Built to demonstrate an interface, not a diagnosis.

NeuroScan does not run any imaging model. It's a portfolio piece exploring how a clinical review tool might be structured, worded and paced — every figure it shows is generated locally in the browser.

No real model

Result figures are derived from a simple hash of the uploaded file, not from any trained imaging network.

Nothing leaves your browser

Patient details and scans are stored only in this browser's local storage — nothing is uploaded to a server.

Clearly labelled

Every results screen carries a disclosure that the output is simulated and not a basis for medical decisions.
